## 무엇을 하는 사이트인가

공공데이터포털 17,163개 오픈API, KOSIS, ECOS, 법령/국회 데이터를 묶어
26개 부처 · 100개 실무 자동화 시나리오를 탐색할 수 있는 웹사이트입니다.

| 페이지 | 내용 |
|--------|------|
| **홈** | 운영 철학, 작동 방식, 유스케이스 카탈로그 (검색 · 필터) |
| **대시보드** | 실거래가 · 미세먼지 · 관보 · 교차분석 라이브 데모 |
| **유스케이스 상세** | 워크플로우, 채팅 데모, 산출물 예시, 절감 효과 |
| **소개** | 파이프라인 구조, 운영 원칙, 기술 스택, 저자 |
