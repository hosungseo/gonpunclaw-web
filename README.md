## 공픈클로는 무엇인가

공픈클로는 공공데이터 오픈API, KOSIS, ECOS, 법령, 국회 데이터를 엮어  
**실무자가 바로 적용할 수 있는 공공업무 자동화 유스케이스 100개**를 정리한 플레이북 플랫폼입니다.

단순히 데이터를 조회하는 데서 끝나지 않고,

- 질문을 받고
- 필요한 근거를 찾고
- 여러 데이터를 교차 해석한 뒤
- 브리핑, 보고서, 점검표 같은 **업무 산출물**로 연결하는 것

을 목표로 합니다.

## 누구를 위한 프로젝트인가

- 공무원
- 공공기관 실무자
- 정책·기획·조직·예산·법제 업무 담당자
- 공공데이터와 AI를 실제 업무에 연결해보고 싶은 사람

---

## 무엇이 다른가

공픈클로는 “예쁘게 답하는 AI”보다  
**검증 가능한 공공업무 자동화**를 지향합니다.

핵심 원칙은 세 가지입니다.

### 1. Research-first
바로 구현하지 않고 먼저 맥락과 근거를 확인합니다.  
법령, API, 데이터 구조를 읽고 시작합니다.

### 2. Verification loop
끝났다고 말하기 전에 build, 응답, 화면, 산출물을 확인합니다.  
완료는 구현이 아니라 검증까지입니다.

### 3. Quality gate
요청 충족, 안전, 재현성, 기록까지 보고 작업을 닫습니다.  
다음 세션이 바로 이어받을 수 있어야 합니다.
