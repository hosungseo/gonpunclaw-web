자동화 개요

청년 대상 복지·고용 서비스 목록 자동 통합 → 연령·지역별 맞춤 안내

직제근거

고용노동부와 그 소속기관 직제 시행규칙 제조 ⑩항: 청년고용기획과장 - 고용노동부 소관 청년정책 시행계획의 총괄ㆍ조정 및 추진실적 점검

사용 API

청년정책 API정부24 공공서비스KOSIS DT_1B04006 (시군구별 청년인구 15~34세)ECOS 청년실업률

워크플로우

1

청년정책 API, 정부24 공공서비스 API 호출 (지역코드(시군구))

2

응답 데이터 파싱 및 정규화 (JSON/XML → 표준 포맷)

3

마크다운 리포트 생성 (핵심 수치 + 표 + 시사점)

4

텔레그램 전송 또는 Google Drive 저장
