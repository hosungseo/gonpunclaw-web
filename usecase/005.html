부처별 기능직위·일반직 정원 비율 분기 분석 (행정안전부 조직국 핵심)

행정안전부행정정보공유과

자동화 개요

기능분류체계 + 직제규칙 + KOSIS 공공행정 취업자 교차 → 기능직위 대 일반직 비율 + 타부처 비교 → 이상치 탐지

직제근거

행정안전부와 그 소속기관 직제 시행규칙 제조 ⑪항: 행정정보공유과장 - 공공분야 개인 맞춤형 데이터 활용 서비스에 관한 기준 및 절차의 수립ㆍ시행

사용 API

기능분류체계 API (15062615, 17,567개)법제처 직제 시행규칙 APIKOSIS 공공행정 취업자 통계 — 정원 수요 교차

워크플로우

1

기능분류체계 API (15062615, 17,567개), 법제처 직제 시행규칙 API API 호출 (조회기간(YYYYMM), 법령MST/검색어)

2

응답 데이터 파싱 및 정규화 (JSON/XML → 표준 포맷)

3

memory/에서 전기 데이터 로드 → 전기 대비 증감 산출

4

이상치/기준 초과 자동 탐지 → 경보 대상 추출

5

마크다운 리포트 생성 (핵심 수치 + 표 + 시사점)

6

텔레그램 전송 또는 Google Drive 저장
